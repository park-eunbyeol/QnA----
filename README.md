
# Diary Web Project (Q&A 기반)

## 프로젝트 소개
Diary Web Project는 PHP와 MySQL을 사용하여 구현한  
개인 기록용 다이어리 웹 프로젝트입니다.  
Q&A 게시판 구조를 응용하여,  
하루의 기록을 작성하고 관리할 수 있는 기능을 구현했습니다.

웹 서비스의 기본적인 CRUD 기능과  
서버–데이터베이스 연동 구조를 학습하는 것을 목표로 개발했습니다.

---

## 주요 기능
- 다이어리 목록 조회
- 다이어리 작성 / 수정 / 삭제 (CRUD)
- 기록 상세 조회
- PHP–MySQL 데이터베이스 연동
- 게시판 형태를 활용한 개인 기록 관리

---

## 기술 스택
- **Backend**: PHP  
- **Database**: MySQL  
- **Frontend**: HTML, CSS, JavaScript  
- **Server**: Apache  
- **Tools**: cursor, Git
---

  ## 구현 방식
- PHP를 활용한 서버 사이드 로직 구현
- MySQL을 통한 데이터 저장 및 조회 처리
- 게시판 구조를 응용한 다이어리 데이터 관리
- Cursor를 활용한 코드 작성 및 기능 구현 보조
- 웹 서비스 기본 요청–응답 흐름 구현

※ 프로젝트는 기능 구현 중심으로 개발되었으며,  
구조는 점진적으로 개선 중입니다.
